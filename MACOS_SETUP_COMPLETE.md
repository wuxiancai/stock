### 方法2：启动完整服务
```bash
# 1. 启动后端API服务器
python3 simple_api.py

# 2. 在新终端启动前端服务器
cd frontend
python3 -m http.server 3000

# 3. 访问 http://localhost:3000
```

### 方法3：一键启动（如果终端支持）

## 🐛 故障排除

### 如果页面无法显示数据
1. 检查浏览器控制台是否有错误
2. 确认Python3已安装：`python3 --version`
3. 尝试直接打开 `frontend/test.html`

### 如果API服务器无法启动
1. 检查端口8080是否被占用：`lsof -i :8080`
2. 尝试使用不同端口
3. 检查Python3安装和权限

## 📞 支持

如果遇到任何问题，请检查：
1. Python3是否正确安装
2. 浏览器是否支持现代JavaScript
3. 网络连接是否正常
