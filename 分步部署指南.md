## 故障排除

### PostgreSQL连接问题

如果遇到PostgreSQL连接问题：

1. **检查服务状态：**
   ```bash
   sudo systemctl status postgresql
   # 或
   brew services list | grep postgresql
   ```

2. **检查认证配置：**
   ```bash
   sudo cat /etc/postgresql/*/main/pg_hba.conf | grep -v "^#"
   ```

3. **手动测试连接：**
   ```bash
   psql -h localhost -U wuxiancai -d stock_trading
   ```

### Redis连接问题

如果遇到Redis连接问题：

1. **检查服务状态：**
   ```bash
   sudo systemctl status redis-server
   # 或
   brew services list | grep redis
   ```

2. **手动测试连接：**
   ```bash
   redis-cli ping
   ```

### 权限问题

如果遇到权限问题：

1. **检查文件权限：**
   ```bash
   ls -la step*.sh
   ```

2. **添加执行权限：**
   ```bash
   chmod +x step*.sh
   ```
