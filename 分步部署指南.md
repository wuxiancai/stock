# 股票交易系统分步部署指南

## 📋 部署概览

本系统采用三步部署方案，确保环境配置、数据库初始化和应用启动的顺序性和可靠性。

## 🚀 部署步骤

### 第一步：环境和依赖安装
```bash
chmod +x step1_environment_setup.sh
./step1_environment_setup.sh
```

**功能说明：**
- ✅ 检测操作系统类型（Ubuntu/CentOS/macOS）
- ✅ 安装系统依赖（curl, wget, git等）
- ✅ 配置Python环境（Python 3.8+）
- ✅ 创建虚拟环境
- ✅ 安装PostgreSQL客户端和开发库
- ✅ 安装Redis客户端
- ✅ 安装Python项目依赖
- ✅ 创建必要目录（logs, uploads, data, backups）

### 第二步：数据库安装和初始化
```bash
chmod +x step2_database_setup.sh
./step2_database_setup.sh
```

**功能说明：**
- ✅ 自动安装PostgreSQL服务器（如果未安装）
- ✅ 智能检测和启动PostgreSQL服务
- ✅ 自动修复权限和配置问题
- ✅ 安装和启动Redis服务器
- ✅ 配置PostgreSQL认证（pg_hba.conf）
- ✅ 创建数据库用户：`wuxiancai`
- ✅ 创建数据库：`stock_trading`
- ✅ 授予用户完整权限
- ✅ 初始化数据库表结构
- ✅ 创建环境配置文件（.env）
- ✅ 测试数据库连接

### 第三步：应用启动
```bash
chmod +x step3_app_startup.sh
./step3_app_startup.sh
```

**功能说明：**
- ✅ 检查虚拟环境和配置文件
- ✅ 验证数据库连接
- ✅ 运行数据库迁移（如果存在）
- ✅ 生成应用管理脚本：
  - `start.sh` - 生产模式启动
  - `start_dev.sh` - 开发模式启动
  - `stop.sh` - 停止应用
  - `status.sh` - 状态检查

## 🎯 一键部署命令

```bash
# 完整部署流程
./step1_environment_setup.sh && ./step2_database_setup.sh && ./step3_app_startup.sh
```

## ⚠️ 注意事项

1. **权限要求**: 部署过程需要sudo权限来安装系统包和配置服务
2. **网络要求**: 需要互联网连接来下载依赖包
3. **系统要求**: 
   - Ubuntu 18.04+ / CentOS 7+ / macOS 10.15+
   - Python 3.8+
   - 至少2GB可用内存
   - 至少5GB可用磁盘空间

4. **安全建议**: 
   - 生产环境请修改默认密码
   - 配置防火墙规则
   - 启用SSL/TLS加密

## 📞 故障排除

如果遇到问题，请检查：

1. **日志文件**: `logs/` 目录下的相关日志
2. **服务状态**: 
   ```bash
   ./status.sh  # 检查应用状态
   sudo systemctl status postgresql  # 检查PostgreSQL状态
   sudo systemctl status redis  # 检查Redis状态
   ```
3. **端口占用**: 
   ```bash
   netstat -tlnp | grep :5432  # PostgreSQL
   netstat -tlnp | grep :6379  # Redis
   netstat -tlnp | grep :8080  # 应用端口
   ```
